# React Commerce donut


## Starting this
- starting front:

``
npm install
``

``
npm dev start
``

- starting server:

`
cd server
`

`
npm i
`

`
npx prisma migrate dev --name init
`

`
npm run start
`
## Building project
if you entered the server then enter: `cd ..`
- building project: `npm run buildd`
